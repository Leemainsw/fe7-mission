# 11.24 미션 제출

## 장바구니 기능이 있는 만두 쇼핑몰 만들기

### 기능구현 요구사항

1. 제품 목록을 화면에 렌더링한다.
2. 각 제품을 클릭 시 장바구니에 추가된다.
3. 장바구니에 담긴 제품 목록을 별도의 페이지(또는 영역)에서 확인할 수 있다.
4. 장바구니의 모든 아이템을 한 번에 삭제할 수 있다.
5. 장바구니의 정보를 localStorage에 저장하여 새로고침 시에도 데이터가 보존된다.
6. 주문하기 버튼을 클릭 시, 장바구니가 초기화된다.
7. 오늘 하루 보지 않기 기능이 있는 팝업(모달) 창을 구현한다.
8. JSON 파일로부터 제품 정보를 불러온다.
